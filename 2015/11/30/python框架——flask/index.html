Python框架——flask

  • Python框架
  • 2015-11-30

一、安装

1. 安装virtualenv

pip install virtualenv

2. 创建虚拟环境

virtualenv --no-site-packages [-p D:\PythonEnvironment\python.exe] E:\interpreter\pyflask

3. 激活环境

cd pyflask/Scripts
activate

4. 退出环境

deactivate

5. flask安装与使用（先激活环境，在执行）

pip install flask

二、启动

1. flask库

应用代码

from flask import Flask
app = Flask(__name__)
app.run([host='0.0.0.0', port='80', debug=True])

启动

python xxx.py


2. flask-script库

安装

pip install flask-script

应用代码

from flask import Flask
from flask_script import Manager
app = Flask(__name__)
manege = Manager(app)
manage.run()

启动

python xxx.py runserver -p -h -d

三、路由规则

route() 装饰器把一个函数绑定到对应的 URL 上

例如

@app.route('/')
def index():
    return 'Index Page'

@app.route('/hello/')
def hello():
    return 'Hello World'


定义格式

<转换器(类型):参数名>
名称 类型
string 字符串，默认类型
int 整型
float 浮点型
path 和默认的相似，但也接受斜线

四、请求

1. 获取参数

GET请求

request.args
request.args['key']
request.args.get('key')

POST请求

request.form
request.form['key']
request.form.get('key')

获取多个同样的key对应的vallue值

getlist('key')


2. 获取请求路径

request.path


3. 请求cookies

request.cookies


4. 获取上传文件

request.files

取值

request.files.get('key')
request.files['key']

五、响应

1. 导包 
from flask import make_response


2. 使用

make_response('响应内容', 响应状态码)

六、cookie与session

1. cookie

设置令牌

res = make_response()
res.set_cookie(key,value,max_age)

删除

res=make_response()
res.delete_cookie(key)


2. session

session数据保存在客户端
导包

from flask import session

使用

session['key']=value

加密复杂程度

secret_key:'xxx'


session数据保存在服务器端
安装

pip install redis flask-session

导包

from flask_session import Session

使用

app.config['SESSION_TYPE'] ='redis'
app.config['SESSION_REDIS'] = redis.Redis()
Session(app)

删除

del session['key']

redirect(url_for('生成蓝图的第一个参数.跳转的函数名称', 参数名=值, 参数名2=值2))

九、模型

1. 安装flask-sqlalchemy库 
pip install flask-sqlalchemy


2. 创建模型

from flask_sqlalchemy import SQLAlchemy
# 生成数据库访问对象db
db = SQLAlchemy()
class 类(db.Model):
    # 定义id主键，自增字段
    id = db.Column(db.Integer, primary_key=True, autoincrement=True)
    # 定义不能为空，且唯一的姓名字段
    name = db.Column(db.String(10), unique=True, nullable=False)
    # 定义整型，默认为20的年龄字段
    age = db.Column(db.Integer, default=20)
    __tablename__ = 'stu'


3. 配置数据库信息

app.config['SQLALCHEMY_DATABASE_URI'] = 'mysql+pymysql://username:password@host:3306/database'
app.config['SQLALCHEMY_TRACK_MODIFICATIONS'] = False
db.init_app(app)


4. 映射

创建

db.create_all()

删除

db.drop_all()


5. 数据处理

增

db.session.add(插入对象)
db.session.add_all(插入对象列表)
db.session.commit()


删

db.session.delete(删除对象)
db.session.commit()


改

db.session.add(插入对象)
db.session.commit()

语法 功能
query 管理器
filter(模型名.字段 == 值) 过滤
filter_by(字段 = 值) 过滤
first() 查询结果中的第一个数据对象
all() 查询结果中的所有数据对象，返回列表

order_by(模型名.字段) 升序
order_by(-模型名.字段) 降序
offset、limit 分页
contains 模糊查询，包含
like：% _ 模糊查询
startswith 模糊查询，开头
endswith 模糊查询，结尾
in_、notin_ 包含
__gt__() 大于

__lt__() 小于

and、or 多条件
and_、or_、not_ 多条件
