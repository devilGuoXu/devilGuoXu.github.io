Python框架——tornado

  • Python框架
  • 2015-11-30

一、安装

1. 安装virtualenv

pip install virtualenv

2. 创建虚拟环境

virtualenv --no-site-packages [-p D:\PythonEnvironment\python.exe] E:\interpreter\pytornado

3. 激活环境

cd pytornado/Scripts
activate

4. 退出环境

deactivate

5. tornado安装与使用（先激活环境，在执行）

pip install tornado

二、Tornado应用

1. 获取tornado.web.Application对象 
参数handlers=[(访问路由, 执行方法类),...]


2. 执行方法类

import tornado.web

继承tornado.web.RequestHandler

定义对应的方法处理对应的请求（如：处理get请求，则定义def get(self)方法）


3. 监听端口

Application对象.listen(端口)


4. 启动

import tornado.ioloop

tornado.ioloop.IOLoop.current().start()

三、命令行

1. 定义默认的监听端口 
from tornado.options import define

define('port', default=8080, type=int)


2. 解析命令行中参数

from tornado.options import parse_command_line

parse_command_line()


3. 获取命令行中的port参数(端口)

from tornado.options import options

options.port


4. 启动命令

python xxx.py --port=端口值

四、路由定义

定义

tornado.web.Application(handlers=[('/days/(\d+)/(\d+)/'), DasyHandler])
或
tornado.web.Application(handlers=[('/days/(?P<month>\d+)/(?P<day>\d+)/'),])

函数

def get(self, month, day)

第一种参数需要依顺序决定，第二种不需要

五、HTTP行为方法

行为方法 功能
GET 只用于获取数据
POST 只用于创建数据
PUT 修改某个对象的全部属性
PATCH 修改某个对象的部分属性
DELETE 删除数据

六、切入点函数

函数 时间点
def initialize() 实例化初始内容，在调用行为方法之前将自动调用
def prepare() 在调用行为方法之前将自动调用
def on_finish() 在调用行为方法之后将自动调用

七、模板

1. 模板文件路径定义 
Application(handlers=[], template_path='指定templates文件夹的路径')


2. 获取templates路径

template_path=os.path.join(os.path.dirname(os.path.abspath(__file__)), 'templates')


3. 父模板

{ % block 名字 % }
{ % end % }

4. 子模板

继承

{ % extends ‘父模板’ % }

5. 语法

解析标签: { % 标签名 % } { % end % }

标签 语法
if标签 { % if 条件 % } { % elif 条件 % } { % else % } { % end % }
for标签 { % for 变量 in [] % } { % end % }
set标签 { % set 变量=值 % }
while标签 { % while 条件% } { % end % }
try标签 { % try % } { % except % } { % finally % } { % end % } 


解析变量:  { { 变量名 } }


6. 静态文件配置与使用

tornado.web.Application(handlers=[], template_path='', static_path='指定static文件夹的路径')


7. 获取static路径

static_path=os.path.join(os.path.dirname(os.path.abspath(__file__)), 'static')


8. 导入静态文件

href='/static/css/xxx.css'
或
{ { static_url('css/xxx.css') } }

八、模型

1. 安装sqlalchemy库 
pip install sqlalchemy


2. 数据库连接配置

db_url = mysql+pymysql://root:123456@localhost:3306/database


3. 创建连接、引擎

from sqlalchemy import create_engine

engine = create_engine(db_url)


4. 模型定义

# 必须继承于Base
Base = declarative_base(bind=engine)


5. 创建会话对象

from sqlalchemy.orm import sessionmaker

DbSession = sessionmaker(bind=engine)
session=DbSession()


6. 增删查改

增

session.add(对象)
session.add_all([对象1, 对象2...])

删

session.delete(对象)
filter(条件).delete()

查

filter(模型名.字段 == 值)
filter_by(字段 == 值)

改

同增的操作
filter(条件).update({'字段1':值, '字段2':值.....})

需要提交事务

session.commit()
