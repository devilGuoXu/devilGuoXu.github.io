Python框架——过滤器

  • Python框架
  • 2015-11-30
过滤器 功能 案例
add 将value的值增加N { { value | add: “N” } }
addslashes 在value中的引号前增加反斜线 { { value | addslashes } }
capfirst value的第一个字符转化成大写形式 { { value | capfirst } }
cut 从给定value中删除所有arg的值 { { value | cut:arg } }
date 格式化时间格式 { { value | date:”Y-m-d H:i:s” } }
default 如果value是False，输出为缺省值，如value是“”，输出nothing { { value | default: “nothing” } }
default_if_none 如果value是None，输出为缺省值，如value是None，输出nothing { { value | default_if_none:”nothing” } }
dictsort 如果value是字典，那返回值是按照关键字排序的结果 { { value | dictsort:”name”} }
dictsortreversed 如果value是字典，那返回值是按照关键字排序的结果的反序 { { value | dictsortreversed:”name” } }
divisibleby 如果value能够被arg整除，那么返回值将是True { { value | divisibleby:arg } }
escape 替换value中的某些字符，以适应HTML格式 { { value | escape } }
filesizeformat 格式化value，使其成为易读的文件大小 { { value | filesizeformat } }
first 返回列表/字符串中的第一个元素 { { value | first } }
iriencode 将value中非ASCII字符转化成URL中编码，已URLENCODE，则无效 { { value | iriencode } }
join 使用指定字符串连接一个list，作用同python的str.join(list) { { value | join:”arg” } }
last 返回列表/字符串中的最后一个元素 { { value | last } }
length 返回value的长度 { { value | length } }
length_is 如果value的长度等于arg的时候返回True { { value | length_is:”arg” } }
linebreaks value中”\n”将被< br/ >替代，并且整个value使用< /p >包围起来 { { value | linebreaks } }
linebreaksbr value中的”\n”将被< br/ >替代 { { value | linebreaksbr } }
linenumbers 显示的文本，带有行数 { { value | linenumbers } }
ljust 在一个给定宽度的字段中，左对齐显示value { { value | ljust } }
center 在一个给定宽度的字段中，中心对齐显示value { { value | center } }
rjust 在一个给定宽度的字段中，右对齐显示value { { value | rjust } }
lower 将一个字符串转换成小写形式 { { value | lower } }
random 从给定的list中返回一个任意的Item { { value | random } }
removetags 删除value中tag1,tag2…的标签 { { value | removetags:”tag1 tag2…” } }
safe 系统设置autoescaping打开的时候，输出不进行escape转换 { { value | safe } }
safeseq 同safe，safe针对字符串，而safeseq针对多个字符串组成的sequence { { value | safeseq } }
slice 与python语法中的slice(切片)相同 { { some_list | slice:”N” } }
striptags 删除value中的所有HTML标签 { { value | striptags } }
time 格式化时间输出 { { value | time[:”H:i”] } }
title 转换一个字符串成为title格式 { { value | title } }
truncatewords 将value切成truncatewords指定的单词数目 { { value | truncatewords:N } }
upper 转换一个字符串为大写形式 { { value | upper } }
urlencode 将一个字符串进行URLEncode { { value | urlencode } }
wordcount 返回字符串中单词的数目 { { value | wordcount } }
