web前端——jQuery

  • web前端
  • 2017-12-28

一、导入jQuery

在script标签的src中导入

js/jquery.min.js
# 或
https://cdn.bootcss.com/jquery/3.3.1/jquery.min.js

二、节点操作

1. 获取节点
语法 返回值
$(‘选择器’) jQuery的节点对象
$(‘选择器:first’) 当前页面中第一个选择器对应的标签
$(‘选择器:last’) 当前页面中最后一个选择器对应的标签
$(‘选择器1 选择器2:first’) 选择器1标签中第一个选择器2对应的标签
$(‘选择器 *:first-child’) 选择器标签中第一个子标签
2. 创建节点
语法 返回值
$(‘html标签语法’) 标签对象
3. 添加节点
语法 返回值
$(‘选择器’).append(标签对象) 在选择器标签的最后添加一个子标签
$(‘选择器’).prepend(标签对象) 在选择器标签的最前面插入一个子标签
$(‘选择器’).before(标签对象) 在选择器标签的前面添加一个同级标签
$(‘选择器’).after(标签对象) 在选择器标签的后面添加一个同级标签
4. 删除节点
语法 返回值
$(‘选择器’).remove() 删除指定选择器对应标签
$(‘选择器’).empty() 清除指定选择器对应标签中的内容

三、属性操作

1. 获取属性
名称 语法
普通 $(‘选择器’).attr(属性名)
innerHTML（标签内容属性） $(‘选择器’).html()
innerText（标签文本内容） $(‘选择器’).text()
value（单标签内容） $(‘选择器’).val()
2. 增加、修改属性
名称 语法
普通 $(‘选择器’).attr(属性名,值)
innerHTML（标签内容属性） $(‘选择器’).html(值)
innerText（标签文本内容） $(‘选择器’).text(值)
value（单标签内容） $(‘选择器’).val(值)
3. class属性
语法 功能
$(‘选择器’).addClass(‘类名’) 给选择器对应标签添加class值
$(‘选择器’).removeClass(‘类名’) 移除指定选择器对应标签的class值
4. 样式属性
语法 功能
$(‘选择器’).css(样式属性名) 获取样式属性值
$(‘选择器’).css(样式属性名, 值) 设置样式
$(‘选择器’).css({属性名:值, 属性名:值 …}) 同时设置多种样式属性

四、事件绑定

$(‘选择器’).on(‘事件名’,函数) —— 同js中的addEventListener

$(‘选择器(父)’).on(‘事件名’,’选择器’(子),函数)

注：js转jQuery：$(js对象)=jQuery对象
