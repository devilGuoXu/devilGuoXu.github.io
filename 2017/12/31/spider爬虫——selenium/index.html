spider爬虫——selenium

  • spider爬虫
  • 2017-12-31

一、安装

  1. 安装包

    pip install selenium

  2. 安装浏览器驱动

    chrome谷歌：http://chromedriver.storage.googleapis.com/index.html

    Firefox火狐：https://github.com/mozilla/geckodriver/releases/

    IE：http://www.nuget.org/packages/Selenium.WebDriver.IEDriver/

    最后将下载文件放入/tools中

  3. 配置环境变量

    进入配置文件sudo vim ~/.bash_profile

    添加path变量目标目录PATH=$PATH:......:/tools

    查看当前变量值echo $PATH

二、启动

启动配置

三、元素定位

描述 定位一个元素 定位多个元素
元素id定位 find_element_by_id find_elements_by_id
元素name定位 find_element_by_name find_elements_by_name
元素class定位 find_element_by_class_name find_elements_by_class_name
css选择器定位 find_element_by_css_selector find_elements_by_css_selector
标签定位 find_element_by_tag_name find_elements_by_tag_name
xpath表达式定位 Find_element_by_xpath find_elements_by_xpath
完整超链接(文字)定位 find_element_by_link_text find_elements_by_link_text
部分链接定位 find_element_by_partial_link_text find_elements_by_partial_link_text

四、控制浏览器常见方法

方法 描述
set_window_size() 设置浏览器的大小
back() 控制浏览器后退
forward() 控制浏览器前进
refresh() 刷新当前页面
clear() 清除文本
send_keys(value) 模拟按键输入
click() 单击元素
submit() 用于提交表单
get_attribute(name) 获取元素属性值
is_displayed() 设置该元素是否用户可见
location 定位元素坐标
size 返回元素的尺寸
text 获取元素的文本
quit() 关闭窗口

五、鼠标事件

方法 描述
ActionChains(driver) 构造ActionChains对象
move_to_element(above) 执行鼠标悬停操作
click() 单击
click_and_hold() 左击
double_click() 双击
context_click() 右击
drag_and_drop() 拖动
perform() 执行所有 ActionChains 中存储的行为，可以理解成是对整个操作的提交动作

六、键盘事件

模拟键盘按键 描述
send_keys(Keys.BACK_SPACE) 删除键（BackSpace）
send_keys(Keys.SPACE) 空格键（Space）
send_keys(Keys.TAB) 制表键（Tab）
send_keys(Keys.ESCAPE) 回退键（Esc）
send_keys(Keys.ENTER) 回车键（Enter）
send_keys(Keys.CONTROL,‘a’) 全选（Ctrl+A）
send_keys(Keys.CONTROL,‘c’) 复制（Ctrl+C）
send_keys(Keys.CONTROL,‘x’) 剪切（Ctrl+X）
send_keys(Keys.CONTROL,‘v’) 粘贴（Ctrl+V）
send_keys(Keys.F1…Fn) 键盘 F1…Fn

七、多表单切换

方法 描述
switch_to.frame() 将当前定位的主体切换为frame/iframe表单的内嵌页面中
switch_to.default_content() 跳回最外层的页面

八、多窗口切换

方法 描述
current_window_handle 获得当前窗口句柄
window_handles 获得当前所有窗口的句柄
switch_to.window() 切换到相应的窗口

九、警告框

方法 描述
text 返回 alert/confirm/prompt 中的文字信息
accept() 接受现有警告框
dismiss() 解散现有警告框
send_keys(keysToSend) 发送文本至警告框。keysToSend：将文本发送至警告框。

十、下拉框

方法 描述
select_by_value(‘选择值’) 相当于我们使用鼠标选择下拉框的值

十一、JavaScript

方法 描述
execute_script(‘window.scrollTo(左边距,上边距);’) 将滚动条拉至对应坐标

十二、窗口截图

方法 描述
get_screenshot_as_file(filename) 用于截取当前窗口，并把图片保存到本地
get_screenshot_as_png() 图片保存到本地，借助PIL包，只能png格式

十三、cookie

方法 描述
get_cookies() 获得所有cookie信息
get_cookie(name) 返回字典的key为“name”的cookie信息
add_cookie(cookie_dict) 添加cookie。“cookie_dict”指字典对象，必须有name 和value 值
delete_cookie(name,optionsString) 删除cookie信息。“name”是要删除的cookie的名称，“optionsString”是该cookie的选项，目前支持的选项包括“路径”，“域”
delete_all_cookies() 删除所有cookie信息

十四、文件上传

十五、关闭浏览器

方法 描述
close() 关闭单个窗口
quit() 关闭所有窗口
